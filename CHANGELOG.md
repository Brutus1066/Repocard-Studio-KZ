# Changelog

All notable changes to RepoCard Studio will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-21

### Added
- Initial public release
- GitHub repository metadata fetching from public URLs
- Three SVG card templates: Modern, Minimal, and Gradient
- README snippet generator with customizable sections
- Release notes draft generator from recent commits
- Press kit export with complete share-kit folder structure
- PNG export at 1200Ã—630 (optimal for social sharing)
- Optional attribution toggle
- Custom color picker for gradient templates
- Windows installer (MSI and NSIS)

### Security
- No telemetry or analytics
- No cloud storage - all processing is local
- GitHub tokens are optional and never logged
